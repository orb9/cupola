X25519 keys are the standard age recipient. Post-quantum hybrid keys add an ML-KEM-768 layer so files stay safe even against a future quantum computer — they're larger, but decrypt just the same.

Import an identity

Paste an existing age secret key (starts with AGE-SECRET-KEY-1 or AGE-SECRET-KEY-PQ-1). The matching public recipient is derived locally.

Saved keys

Secret keys are stored in this browser's local storage, unencrypted. Don't use this on a shared or public computer, and export a backup somewhere safe.
